# Mini Projects

This folder contains the mini projects completed for the **Introduction to RDBMS** module.

## 1. Normalization Assignment

### Objective

Convert the given tables into:

* First Normal Form (1NF)
* Second Normal Form (2NF)
* Third Normal Form (3NF)

### Learning Outcomes

* Understand atomic values and 1NF
* Remove partial dependency to achieve 2NF
* Remove transitive dependency to achieve 3NF
