# Introduction to SQL

This folder contains my SQL learning journey as part of the Wipro TalentNext program. It includes notes, hands-on practice, assessments, and SQL scripts covering fundamental to intermediate SQL concepts.

---

## Topics Covered

- Introduction to SQL
- Retrieving Data Using the SQL SELECT Statement
- Restricting and Sorting Data
- Using Single-Row Functions to Customize the Output
- Reporting Aggregated Data Using Group Functions
- Using Set Operators
- Displaying Data from Multiple Tables Using Joins

---

## Each Topic Contains

- SQL Scripts (.sql)
- Hands-on Assessments
- Query Output Screenshots

---

## Software Used

- Oracle Database 11g Express Edition (XE)
- Oracle SQL Developer

---

## Learning Objective

The objective of this module is to develop practical SQL skills including data retrieval, filtering, sorting, aggregation, joins, and writing efficient SQL queries using Oracle SQL.
